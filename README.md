## About

Brain Games is a set of 5 games to train your brain! The games are:
1. brain-calc
2. brain-even
3. brain-gcd
4. brain-prime
5. brain-progression

## Setup

```sh
$ git clone https://github.com/davydovks/php-project-45.git

$ cd php-package-45

$ make install
```

## Run games

You can start a game using any of these commands:
```sh
$ make brain-calc

$ make brain-even

$ make brain-gcd

$ make brain-prime

$ make brain-progression
```
